# DepthX – Conceptual Debt Detection Web App

## Overview
DepthX is an educational web app used to identify the areas where student lack by giving short quiz and analysing the result and identify the areas which require improvement.

---

## Problem Statement
Teams must build a creative software solution (web or mobile) that addresses a real educational need. The goal is to enhance some aspect of learning or teaching using code. Projects should be practical and directly benefit 
education; creative ideas are encouraged so long as they are technically feasible.

---

## Features
- Topic selection interface
- Quiz-based assessment
- Score evaluation
- Detection of areas on conceptual debt
- Result and feedback page
- Progress saved using browser LocalStorage
- Clean and intuitive UI

---

## Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage

> No backend, database, AI, or machine learning is used.

---

## How to Run the Application
1. Download or clone the repository
2. Open the project folder
3. Double-click 'index.html'
4. The app will open in your web browser

No setup or installation is required.
